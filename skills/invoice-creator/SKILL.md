# Skill — Invoice Creator

## Description
Création de factures et devis professionnels pour l'agence.

## Déclencheurs
- "crée une facture pour [client]"
- "génère un devis"
- "facture [montant] à [client]"

## Méthode
1. Collecter infos client (nom, adresse, TVA si applicable)
2. Détailler les prestations
3. Appliquer TVA belge (21%) si applicable
4. Générer PDF professionnel
5. Envoyer par email si demandé

## Contraintes
- Numérotation séquentielle
- Mentions légales belges obligatoires
- Délai de paiement : 30 jours
- Coordonnées Ayman sur chaque document
